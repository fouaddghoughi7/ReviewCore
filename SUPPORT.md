# Support

ReviewCore is in the foundation stage. Support is focused on repository setup,
architecture, documentation, and contribution workflow.

## Where To Ask

- Use GitHub issues for reproducible bugs, documentation problems, and scoped
  development questions.
- Use pull requests for proposed documentation or code changes.
- Use [Security Policy](SECURITY.md) for vulnerabilities.

## Before Asking

- Read [README](README.md).
- Read [Documentation Index](docs/README.md).
- Search existing issues and pull requests.
- Include the ReviewCore version, PHP version, Composer version, operating
  system, and steps to reproduce when reporting a problem.

## Current Limits

Business feature support, production deployment support, and commercial support
are not defined during the `v0.1.0` foundation stage.
