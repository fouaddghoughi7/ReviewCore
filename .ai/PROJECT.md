# ReviewCore

## Vision

ReviewCore is an enterprise-grade AI-powered review management platform.

It is designed to help businesses, agencies, bloggers, affiliate marketers, and WordPress users create, manage, optimize, and analyze reviews using artificial intelligence.

The platform focuses on high-quality reviews, SEO, E-E-A-T, automation, and extensibility.

---

# Mission

Build the most modular, extensible, and developer-friendly review platform.

Every feature should be implemented as an independent package with minimal coupling.

---

# Non Goals

ReviewCore is NOT:

- A CMS
- A Laravel replacement
- A generic PHP framework
- A WordPress theme
- A simple review plugin

It is a modular review platform.

---

# Coding Philosophy

Always prefer:

- Simplicity
- Readability
- Maintainability
- Extensibility
- Testability

Never sacrifice architecture for convenience.

---

# Development Rules

Every new feature must:

- Include documentation.
- Include tests.
- Respect package boundaries.
- Follow SOLID.
- Follow PSR-12.
- Avoid tight coupling.

No business logic should exist inside infrastructure packages.
